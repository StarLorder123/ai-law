- 这个文档是用于说明对于案件文书、法律法规相关数据处理的一些内容的。

# 案件文书部分

- 目前将案件文书部分提取出来之后的json格式为：

``` json
{
    "wsmc":"xxx", // 文书名称
    "ah":"xxx", // 案号
    "ajlx":xxx, // 案件类型
    "ssdq":"xxx", //诉讼地区
    "ajmc":"xxxx", // 案件名称
    "sj":"xxxx", // 时间
    "fymc":"xx", // 法院名称
    "slcx":"xxx", // 审理程序
    "dsr":[], // 当事人
    "content":{
        "ay"："xxx", // 案由
        "wsnc":"xxxx", // 文书内容
        "flyj":"xxx" // 法律依据
    },
    "extension":{
        "link":"xxx", // 链接
        "filename":"xxxx" // 文件名称
    }
}
```
